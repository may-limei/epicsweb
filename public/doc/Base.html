1.安装依赖包

使用 ESSS 的 Jeong han Lee的依赖包安装脚本。下载和使用方法参考地址：

https://github.com/jeonghanlee/pkg_automation.git

2.下载base 包：

例如 base-3.16.1.tar.gz

3. 解压到EPICS根目录

环境变量 EPICS_BASE指向该目录如：export EPICS_BASE=/usr/local/epics/base-3.16.1

tar -zxvf base-3.16.1.tar.gz  -C /usr/local/epics

4. 在base文件夹中编译

make

5.设置环境变量：

export EPICS_BASE=/usr/local/epics/base-3.16.1
       export EPICS_ARCH=linux-x86_64
       export EPICS_CA_ADDR_LIST=127.0.0.1:5064
       export EPICS_CA_AUTO_ADDR_LIST=no
       export NODE_EPICS_LIBCA=/usr/local/epics/base-3.16.1/lib/linux-x86_64/libca
       export PATH=${EPICS_BASE}/bin/${EPICS_ARCH}:${PATH}

可以把这些命令拷贝到/etc/profile.d/epics.sh文件中（自定义文件）

6. 检查测试

确定能访问EPICS命令

makeBaseApp.pl

有此命令说明EPPICS Base环境设置好了, EPICS命令已经加入到PATH路径
