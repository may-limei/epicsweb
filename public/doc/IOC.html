1.生成 并进入文件夹

$makedir test
    $ cd test

2. 通过拷贝example 模板,构建IOC 应用test 

$makeBaseApp.pl -t example   test

    注:文件夹下生成 文件夹/configure, /testApp和脚本Makefile

$makeBaseApp.pl -i -t  example test

    注:文件夹下增加生成 iocBoot 启动目录.期间提示你本启动脚本启动的IOC应用名称 填写test

3. 编译应用

$make

     注:新生成 /bin /lib /include /db 和/dbd文件夹  

4. 运行程序:

$cd iocBoot/ioctest
    $chmod 755 st.cmd
    ./st.cmd

    注:程序进入iocShell: epics>

5. 察看变量列表

epics> dbl
    $chmod 755 st.cmd
    $./st.cmd

    注: IOC默认启动端口是5064, 如果需要在局域网访问本IOC实例的变量,需要设置linux 环境变量 EPICS_CA_ADDR_LIST 和  EPICS_CA_AUTO_ADDR_LIST.

          终端不能关闭,终端关闭IOC实例停止运行

6. 本机终端测试: 

$ export EPICS_CA_ADDR_LIST="127.0.0.1:5064"
    $ export EPICS_CA_AUTO_ADDR_LIST="NO"
    $ camonitor PVName

          注:此处PVName替换为你的IOC 变量,如 hanlfHost:ai1

$camonitor   hanlfHost:ai1
